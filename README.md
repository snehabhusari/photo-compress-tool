# Image Compressor tool

A fast, free online tool to compress images for social media, messaging, and more.

## Features

- 🖼️ Upload and compress images instantly
- 📊 See before/after file sizes
- 🎚️ Adjust compression quality with a slider
- ⬇️ Download compressed images
- 🚀 Works entirely in your browser (no server needed)
- 📱 Mobile-friendly design

## Why I Built This

I needed a simple, free way to compress images before uploading to social media and messaging apps. Existing tools were either slow, had ads, or required payment. This tool does one thing well: compress images fast and free.

## Tech Stack

- Hono (lightweight web framework)
- HTML5 Canvas API (for image compression)
- Vanilla JavaScript
- Tailwind CSS (styling)

## Deployment

Deployed on Vercel's free plan.
